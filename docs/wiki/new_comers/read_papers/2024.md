# 如何阅读文献

!!! tip "课程信息"
    **主讲人**: 厦门大学 张泽华

    录制于 2024-08。本课程主要介绍了如何查找文献、进行文献管理、文献阅读需要注意什么地方及文献汇报需要注意的点。详细介绍了x-mol、web of science等文献查找网站的应用，endnote和zotero两类文献管理工具的应用等。
